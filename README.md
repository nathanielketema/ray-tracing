# Ray Tracing in Zig

## Getting Started

1. Clone the Repository

   ```console
   git clone https://github.com/nathanielketema/ray-tracing.git
   ```
2. Build and Run

   ```zig
   zig build run
   ```

## Tools used

- [Portable Pixmap Format (PPM)](https://en.wikipedia.org/wiki/Netpbm)
